# Kudelma

Kudelma is a full-stack web application that weaves art, music, books, and recipes into a tapestry
inspired by the current weather and season. Users can discover personalised recommendations and
share their favorite combinations by creating postcard. 💌

The application is available at https://kudelma.vercel.app/.

## Tech Stack

| Layer | Technologies |
|-------|--------------|
| **Frontend** | React (Vite), TypeScript, Tailwind CSS, Redux Toolkit, Axios |
| **Backend** | Node.js, Express, TypeScript, PostgreSQL, Prisma ORM, Redis, Zod |
| **APIs** | Spotify, Finna, Open Library, OpenWeather, ColorMagic, Google Gen AI |
| **Development** | Docker Compose, Nginx |
| **Deployment** | Vercel, Render, Supabase PostgreSQL, Upstash Redis |
