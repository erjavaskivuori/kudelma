# User Instructions

Kudelma requires the user's location to provide recommendations. The user's location is used to
fetch local weather information, which, together with the current season, influences the
recommended artworks, books, recipes, and the application's visual theme.

To receive music recommendations, the user must connect their Spotify account.

The user can browse recommendations and save favorites by clicking the heart icon on an artwork,
book, and recipe, and the "select" button on music items.

To create postcards, the user must create an account. Each postcard features the selected artwork on
the front and can optionally include a music recommendation, book, and recipe on the backside.

Public postcards created by all users can be viewed on the Community page. Each user's own public
postcards are available on their profile page.

The user can delete their own postcards by flipping the postcard (clicking it) and selecting the
trash can icon.

Account settings are available from the user's profile page. There, the user can:

* choose whether their postcards are public
* delete their account

## Getting Spotify recommendations

When user has connected their account they can influence the music recommendations by sharing
what they are doing and what is their mood. If user clicks "Skip for now" the recommendations
will be based only on the weather and seasonal information.
